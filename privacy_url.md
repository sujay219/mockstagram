# Privacy Policy – Bello Todo

Bello Todo respects your privacy.

## Information Collection and Use
Bello Todo collects and stores the following information solely to provide its core functionality:
- Tasks, task metadata, and user preferences created within the extension

The extension does not require user accounts and does not support Google Sign-In.

## Data Storage
All task data and preferences are stored locally in the user’s browser using Chrome storage.
No task data is transmitted to external servers.

## Data Sharing
Bello Todo does not sell, rent, or share user data with third parties.
No data is used for advertising, analytics, or tracking.

## Website Access
The extension injects a user-controlled overlay interface into web pages to display tasks and focus tools.
The extension does **not** read, collect, monitor, or transmit website content, URLs, or browsing activity.

## Security
Reasonable measures are taken to protect user data. No sensitive personal information is intentionally collected.

## Changes
This privacy policy may be updated from time to time. Any changes will be reflected on this page.
